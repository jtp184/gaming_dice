value_string ()

Uses the value to return a textual representation. Converts values outside of (2..10) to face card names.

[show source] 
# File lib/gaming_dice/card.rb, line 86
def value_string
        case value
        when 1
                'Ace'
        when 2..10
                value.to_s
        when 11
                'Jack'
        when 12
                'Queen'
        when 13
                'King'
        when 14
                'Joker'
        else
                'Fool'
        end
end
